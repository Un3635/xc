What is XToken?

XToken is a new digital token wallet, providing users with one-stop storage and transfer services. XToken supports ETH and digital token based on ERC20, which shall also continue to support more mainstream digital tokens.

Does XToken support multiple wallet accounts?

Yes, XToken supports multiple wallet accounts. Click "Me" -> Managing Wallet, and select Import Wallet, inputting the backup mnemonic words of wallet, keystore or private key. After successful verification, set the transaction password, and the wallet account is successfully imported.

Which digital token does XToken support?

It currently supports ETH and token based on ERC20, and shall continue to support more mainstream digital tokens afterward.

Does XToken charge? And what is the rate?

XToken does not charge any fees for the digital token storage and transfer function provided to users. The transfer poundage has also known as the transaction fee is paid to the network node which maintains the operation of the digital token system. The transfer page shows the reference poundage. And the actual poundage incurred actually shall be subject to the actual situation, which is checked on the detailed page of the transaction record.

How to create a wallet?

Open XToken, and select Create a Wallet on the wallet asset page, after setting the transaction password, then the wallet shall be created successfully The user needs to backup the wallet mnemonic words according to the tips, and possessing the mnemonic words shall completely control the wallet assets. XToken does not store users’ wallet mnemonic words, thus, losing mnemonic words means losing wallet assets. Please make sure to keep it properly.

How to import a wallet?

Click "Me" -> Managing Wallet, and select Import Wallet, input the backup wallet mnemonic words. After successful verification, set the transaction password, thus, the wallet is successfully imported. And then you can start managing the backup wallet assets through the mnemonic words, including ETH and contract tokens based on ERC20, etc.

How to transfer/collect payment?

On the page of XToken asset list that select the asset type to be traded is to start the transfer and collect payment: Select Transfer, input the payee’s address (which can be completed by scanning the QR code) and the transfer amount, and the transfer shall be completed after the verification of transaction password. Select the Collect Payment, with the page displaying the QR code of collecting payment, and the payer can scan the QR code to complete the transfer. Or the payee can click the QR code to copy the collecting payment address, sharing and sending to the payer.

How to modify the password? What if you forget the password?

Through setting - Modify password, and the new password can be set after the verification of the original password. Because XToken does not store the user’s password, if you forget the password, the user can only click "Me" -> Managing Wallet, select Delete Wallet and then re-import the wallet. If you forget your password and lose your wallet mnemonic words, then you shall lose your digital assets.

Can the backup wallet mnemonic words be retrieved if they are lost?

XToken does not store the user's wallet mnemonic words, which is only stored on the user's mobile phone. Please keep it properly!

Can the type of digital token displayed on the wallet homepage be modified?

After clicking the Add Asset button on the right side of the wallet homepage and entering the page of Add Assets, select the token type you want to display on the homepage and turn the display switch on or off.

Shall the corresponding assets be lost after checking the digital token displayed on the wallet homepage?

No, it won’t. On the Add Assets page, turning off the display switch for some digital token just means it shall not display on the homepage. If there is the balance of the corresponding digital token, then the asset shall not be lost. After re-opening the display switch, such digital token and corresponding balance shall be displayed on the wallet homepage.

Why the collecting address of ETH and that of token based on ERC20 the same?

The token based on ERC20 is a smart contract token issued on the Ethereum Platform. The ETH address represents an account of the platform. And such account can receive all contract asset types on this platform, so the address is the same as that of ETH.

Why does ETH equal the poundage charged for the token based on ERC20 at the time of transfer?

For the token based on ERC20, the transfer consumes the computing resources of the Ethereum Platform, which needs token ETH of Ethereum Platform as the poundage to be paid to the network node that maintains the Ethereum.
