XToken是什麼？

XToken是一款全新的數位貨幣錢包，為使用者提供一站的存儲和轉帳服務。XToken支援ETH和基於ERC20的數位貨幣，後續將陸續支援更多主流數位貨幣。

XToken支持多個錢包帳戶嗎？

XToken支援多個錢包帳戶，點擊 "我" ->管理錢包，選擇導入錢包，將備份的錢包助記詞、keystore或私密金鑰輸入，驗證成功後，設置交易密碼，錢包帳戶導入成功

XToken支援哪些數位貨幣？

當前支持ETH和基於ERC20的代幣，後續將支援更多主流數位貨幣。

XToken收費嗎？費率是多少？

XToken對於給使用者提供的數位貨幣存儲、轉帳功能，不收取任何費用，轉帳時的手續費是支付給維護數位貨幣系統運行的網路節點，也稱礦工費。轉帳頁面顯示的是參考手續費，實際發生手續費以實際發生為准，具體在交易記錄詳情頁面查看。

如何創建錢包？

打開XToken，在錢包資產頁選擇創建錢包，設置交易密碼後，錢包即創建成功；使用者需要根據提示備份好錢包助記詞，擁有助記詞即可完全控制錢包資產，XToken不存儲使用者錢包助記詞，丟失助記詞即丟失錢包資產，請務必妥善保存。

如何導入錢包？

點擊 "我" ->管理錢包，選擇導入錢包，將備份的錢包助記詞輸入，驗證成功後，設置交易密碼，錢包導入成功，即可開始管理通過該助記詞備份的錢包資產，包括 ETH及基於ERC20的合約代幣等

如何進行轉帳/收款？

在XToken資產清單頁面，選擇想要交易的資產類型，即可開始轉帳收款：選擇轉帳，輸入收款人位址（可通過掃描二維碼完成）和轉帳金額，驗證交易密碼後，即完成轉帳；選擇收款，頁面展示收款二維碼，付款方可以掃描二維碼完成轉帳，或者收款方點擊二維碼複製收款地址，分享發送給付款方。

如何修改密碼？忘記密碼怎麼辦？

通過設置-修改密碼，驗證原密碼後，即可設置新密碼；因為XToken不存儲使用者密碼，如果忘記密碼，使用者只能點擊“我”->管理錢包，選擇刪除錢包，然後重新導入錢包。如果忘記密碼且丟失了錢包助記詞，你將丟失數字資產。

備份的錢包助記詞丟失後能找回嗎？

XToken不存儲使用者的錢包助記詞，助記詞只存儲在使用者手機上，請妥善保管！

錢包首頁顯示的數位貨幣類型可以修改嗎？

通過點擊錢包首頁右側的添加資產按鈕，進入添加資產頁面後，選擇想要在首頁展示的貨幣種類，打開或者關掉展示開關。

勾選掉錢包首頁展示的數字貨幣，對應的資產會丟失嗎？

不會。在添加資產頁面關掉某些數位貨幣的展示開關，只是在首頁不進行展示，如果對應的數字貨幣有餘額，資產不會丟失，重新打開展示開關之後，會在錢包首頁展示該種數位貨幣及對應餘額。

為什麼ETH和基於ERC20的代幣的收款位址是一樣的？

基於ERC20的代幣是在乙太坊平臺發行的智慧合約代幣，ETH位址代表該平臺的一個帳戶，該帳戶可以接收平臺上的所有合約資產類型，所以位址和ETH是一樣的。

為什麼基於ERC20的代幣轉帳時收取的手續費是ETH？

對基於ERC20的代幣，轉帳消耗乙太坊平臺的運算資源，需要用乙太坊平臺的代幣ETH作為手續費，支付給維護乙太坊的網路節點。
