XToken是什么？

XToken是一款全新的数字货币钱包，为用户提供一站的存储和转账服务。XToken支持ETH和基于ERC20的数字货币，后续将陆续支持更多主流数字货币。

XToken支持多个钱包账户吗？

XToken支持多个钱包账户，点击 "我" ->管理钱包，选择导入钱包，将备份的钱包助记词、keystore或私钥输入，验证成功后，设置交易密码，钱包账户导入成功

XToken支持哪些数字货币？

当前支持ETH和基于ERC20的代币，后续将支持更多主流数字货币。

XToken收费吗？费率是多少？

XToken对于给用户提供的数字货币存储、转账功能，不收取任何费用，转账时的手续费是支付给维护数字货币系统运行的网络节点，也称矿工费。转账页面显示的是参考手续费，实际发生手续费以实际发生为准，具体在交易记录详情页面查看。

如何导入钱包？

点击 "我" ->管理钱包，选择导入钱包，将备份的钱包助记词输入，验证成功后，设置交易密码，钱包导入成功，即可开始管理通过该助记词备份的钱包资产，包括 ETH及基于ERC20的合约代币等

如何进行转账/收款？

在XToken资产列表页面，选择想要交易的资产类型，即可开始转账收款：选择转账，输入收款人地址（可通过扫描二维码完成）和转账金额，验证交易密码后，即完成转账；选择收款，页面展示收款二维码，付款方可以扫描二维码完成转账，或者收款方点击二维码复制收款地址，分享发送给付款方。

如何修改密码？忘记密码怎么办？

通过设置-修改密码，验证原密码后，即可设置新密码；因为XToken不存储用户密码，如果忘记密码，用户只能点击“我”->管理钱包，选择删除钱包，然后重新导入钱包。如果忘记密码且丢失了钱包助记词，你将丢失数字资产。

备份的钱包助记词丢失后能找回吗？

XToken不存储用户的钱包助记词，助记词只存储在用户手机上，请妥善保管！

钱包首页显示的数字货币类型可以修改吗？

通过点击钱包首页右侧的添加资产按钮，进入添加资产页面后，选择想要在首页展示的货币种类，打开或者关掉展示开关。

勾选掉钱包首页展示的数字货币，对应的资产会丢失吗？

不会。在添加资产页面关掉某些数字货币的展示开关，只是在首页不进行展示，如果对应的数字货币有余额，资产不会丢失，重新打开展示开关之后，会在钱包首页展示该种数字货币及对应余额。

为什么ETH和基于ERC20的代币的收款地址是一样的？

基于ERC20的代币是在以太坊平台发行的智能合约代币，ETH地址代表该平台的一个账户，该账户可以接收平台上的所有合约资产类型，所以地址和ETH是一样的。

为什么基于ERC20的代币转账时收取的手续费是ETH？

对基于ERC20的代币，转账消耗以太坊平台的运算资源，需要用以太坊平台的代币ETH作为手续费，支付给维护以太坊的网络节点。
